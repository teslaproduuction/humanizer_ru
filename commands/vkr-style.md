# VKR Style Command

Transform text into formal academic style suitable for graduation theses (ВКР), dissertations, and technical documentation.

## Usage
```
/vkr-style [your text]
```

## What it does
- Applies formal academic Russian language style
- Structures content according to ВКР standards
- Uses appropriate technical terminology
- Ensures logical flow and academic rigor
- Formats according to GOST requirements

## Examples
```
/vkr-style Я сделал программу которая собирает данные с датчиков
```

Output:
> В рамках данной работы была разработана программная система для сбора данных с измерительных датчиков...
